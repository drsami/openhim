OpenHIM

OpenHIM is a open source Health Information Mediator built to facilitate interoperability between disparate Health Information Systems.

Download this project as a .zip file Download this project as a tar.gz file

The Open Health Information Mediator (OpenHIM) is a middleware system based on an ESB architecture that aims to enable easier interoperability between disparate Health Information Systems.

We plan to build a community of implementers and developers that support this work so that we can ease the burden of interoperability and enable information sharing between clinicians in real world projects.

The reference implementation of this is for the Rwandan Health Information Exchange (RHIE). The project is know as the RHEA (Rwandan Health Enterprise Architecture) project. Please see the RHEA wiki for more details:

  • Health Information Mediator Documentation
  • RHEA Implementation wiki

Community

If you would like to get involved with this project, more technical developer discussions will take place on the OpenHIM Developer mailing list (https://groups.google.com/d/forum/openhim-dev).

We hope to hear from you soon!
